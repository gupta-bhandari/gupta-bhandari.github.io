  • Articles Recommendation System Using Non-Negative Matrix Factorization

    This project aims to develop a recommendation system for Medium articles using the Non-Negative Matrix Factorization (NMF) technique to group articles with similar topics. NMF is a machine learning method utilized for dimensionality reduction and feature extraction in linear algebra and data analysis, often applied in data preprocessing, image processing, and text mining, where data can be represented as non-negative values. Prior to implementing NMF, several Natural Language Processing (NLP) steps are implemented, including vectorizing the data using TF-IDF. Additionally, Exploratory Data Analysis (EDA) is conducted before applying machine learning techniques. The results will be sorted based on the number of claps (likes) to recommend high-quality articles on similar topics to readers. NLP is specifically employed for text analysis and topic modeling, while TF-IDF (Term Frequency-Inverse Document Frequency) provides a numeric representation of words within a document collection in the context of information retrieval and text mining. Research Paper

Plant Diseases Detection System Using Deep Learning

Farmers traditionally detect plant diseases by observing spots and yellowing on leaves based on their past experience. To enhance this process, a technology has been developed that allows farmers to upload images of leaves to receive insights regarding their health and potential solutions if issues are identified. This project utilizes Convolutional Neural Networks (CNN), a class of artificial neural networks, trained with approximately 4,500 leaf images across four different classes from the PlantVillage dataset. The images feature single leaves with a homogeneous background, free from extraneous noise such as mud or stones. Recognizing that real-world images may differ, the YOLO algorithm was employed to identify regions of interest (ROI) and feed these into the CNN model for prediction. The proposed CNN model consists of 10 layers: 2 sequential layers for data preprocessing and augmentation, 3 convolutional layers, 3 pooling layers, 1 fully connected layer, and 1 output layer utilizing a Softmax activation function. The dataset was divided with 60% allocated for training, 20% for validation, and 20% for testing the model. After training the model for 60 epochs, it achieved a validation accuracy of 96% and a test accuracy of 93%.

This research was published in KEC Conference.

Publication Date: 2022
